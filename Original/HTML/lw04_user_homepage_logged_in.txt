<br><br>Essentially what the recommender does is take a whole lot of your movie ratings and a whole lot of other people's movie ratings and builds an AI-powered machine learning fully automated (insert other bullshit people spin these days here) model and hey presto, you've got yourself the perfect film for your {today} night.

<br><br>What do you need to do? Not much to be honest, just rate some films. The model is only as good as the data you give it, so the more films you rate the better. In terms of how you can rate those films you have a couple of options...

<br><br><b>Search</b> for partcular films you loved (or hated).
<br><b>Rate Films</b> will give you a random film to rate. Here, at LMR we know it can be difficult to think of the films you have seen so maybe spend a few minutes swiping through til you find films that ring a bell.
<br><b>Rate New Films</b> gives you the chance to flex your cinephilic muscles by having a look through the films that no one on the database has seen. This is really helpful for the database as it increases the pool of films that can be recommended.

<br><br>If you're a new user, once you have rated a few films - a minimum of 20 is recommended but the more you rate the better your recommendation will be - you'll need to give us a bit of time (say 24 hours) to build your recommendation model.

<br><br><b>Feeling Friendly</b> lets you build a viewing party of other viewers and gives you the film that is most suited to all of you.
<br><b>Feeling Picky</b> let's you put a few filters on the films to make sure that you are getting the best one to suit your current needs. You can pick up to 3 possible genres, specify that you only want to watch films after a certain year (who can be bothered watching the black and whites), the maximum runtime (who has the time to watch The Titanic) and even the minimum IMDB rating (for those who are very risk averse).

  			document.getElementById("mySidebar").style.display = "none";

		function toggleSidebar() {{
  			var sidebar = document.getElementById("mySidebar");
  			var overlay = document.getElementById("myOverlay");
  			if (sidebar.style.display === "block") {{
      				sidebar.style.display = "none";
      				overlay.style.display = "none"; // Set overlay display to "none"
  				}} else {{
      					sidebar.style.display = "block";
      					overlay.style.display = "block"; // Set overlay display to "block"
  				}}
			}}

			// Tabs
		function openCity(evt, cityName) {{
  			var i;
  			var x = document.getElementsByClassName("city");
  			for (i = 0; i < x.length; i++) {{
    				x[i].style.display = "none";
  				}}
  			var activebtn = document.getElementsByClassName("testbtn");
  			for (i = 0; i < x.length; i++) {{
    				activebtn[i].className = activebtn[i].className.replace(" w3-dark-grey", "");
  				}}
  			document.getElementById(cityName).style.display = "block";
  			evt.currentTarget.className += " w3-dark-grey";
			}}

    				x.className += " w3-show";
  				}} else {{
    				x.className = x.className.replace(" w3-show", "");
  					}}
				}}

		function plusDivs(n) {{
  			slideIndex = slideIndex + n;

		function showDivs(n) {{
  			var x = document.getElementsByClassName("mySlides");
  			if (n > x.length) {{slideIndex = 1}}
  			if (n < 1) {{slideIndex = x.length}} ;
  			for (i = 0; i < x.length; i++) {{
    				x[i].style.display = "none";
  				}}
  			x[slideIndex-1].style.display = "block";
			}}

		showDivs(1);

		// Progress Bars
		function move() {{
  			var elem = document.getElementById("myBar");
  			var width = 5;
  			var id = setInterval(frame, 10);
  			function frame() {{
    				if (width == 100) {{
      					clearInterval(id);
    					}} else {{
      					width++;
      					elem.style.width = width + '%';
      					elem.innerHTML = width * 1  + '%';
    					}}
  				}}
